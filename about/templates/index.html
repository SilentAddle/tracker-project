How does it work?

The App.

Feature & Bug Tracker

Got a bug to report? You can do so simply.
You need only Sign Up for an account and post in the BugFix portion of the Feature and Bug Forum.

If you want to submit a feature request, things are handled a little differently.
You still need to Sign Up for an account, but you will need credits to request a feature.

In order to find the further development of the Unicorn Attractor App, we use a credit system.
On signing up, you get 50 credits, at the end of beginning of each month you get topped back up to 50 credits (So don't be afraid to use them).
These credits can be used to upvote the Features and BugFixes that matter to you.
You can pledge any of your credits to a post, so if you think a feature or bug deserves all of your free credits? Pledge away.
However, if you want to request a feature, you will have to spend more.
To request a feature costs 100 credits, credits can be bought from the store at any time.
